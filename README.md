## 🎯 Objetivo del Laboratorio:
Crear páginas web básicas usando HTML y CSS, haciendo uso de tags HTML básicos y aplicando estilos con CSS.

## 📖 Descripción:
En este laboratorio aprendimos a:

Crear páginas web básicas con HTML

Utilizar tags HTML de texto, listas, enlaces, tablas y formularios

Aplicar estilos CSS a elementos HTML

Comprender el uso de selectores CSS

Trabajar con repositorios Git y GitHub
